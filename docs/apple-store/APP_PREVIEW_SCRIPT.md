# App Preview script (20–30 seconds)

Use this on **Mac (Simulator)** or **iPhone** for the final App Preview. Total length: about **20–30 seconds** at a comfortable pace. No voice-over required; on-screen text is the product.

## Honest one-line (use in description / optional end card)

Cleaning Timeclock helps cleaning teams manage assigned jobs, shift start and stop, and job-site verification.

## Beat sheet (in order)

1. **Login screen** (2–3s)  
App opens on login. Optional: type demo credentials (no real secrets in recordings).
2. **Worker sees assigned job** (3–4s)  
Worker home / dashboard showing at least one assigned job.
3. **Accept job** (2–3s)  
Open job, tap accept (or equivalent).
4. **Start shift with location verification** (4–5s)  
Start shift flow; show location or site check as it appears in the app (OS permission prompt only if you want reviewers to see the flow; avoid exposing real addresses in test data).
5. **Stop shift** (3–4s)  
End the active shift, confirm if applicable.
6. **Admin: jobs / workers overview** (4–5s)  
Log in or switch to admin (separate run or same demo with role switch if the app allows). Show a screen that shows jobs and/or workers at a glance.
7. **End card** (3s)  
Full-screen or clean overlay:

   **Cleaning Timeclock**  
*Workforce time tracking for cleaning teams*

   Subline (if space): *Cleaning Timeclock helps cleaning teams manage assigned jobs, shift start and stop, and job-site verification.*

   ## Production notes

* Use **portrait** unless your listing targets landscape for iPad only.
* Trim silence; avoid scrolling jitter; use stable Wi-Fi or offline-safe flows if the app allows.
* Match the resolution App Store Connect expects for the device family you are targeting for the video slot.
